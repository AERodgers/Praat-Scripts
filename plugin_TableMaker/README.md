# TableMaker - a Praat plug-in for nexted annotations and acoustic data

This allows you to generate elegant plots from data in Praat.

To use it, you need to [download Praat](https://www.praat.org) onto your computer or make sure you have the latest version (at least 6.4.6). Then:
1. Click on the **clone or download** button above followed by **download ZIP**.
2. Download and unzip the file.
3. Copy the **plugin_TableMaker-master** folder and *all* its contents into the **Praat preferences folder**.

    ([Click on this link](https://praat.org/manual/preferences_folder.html) for guidelines on how to locate this folder on your computer.)
